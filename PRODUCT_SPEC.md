# Product Spec

RankForge helps small teams and creators audit websites, connect search performance data, and turn technical issues into clear next actions.

## v1 Outcomes

- Add a workspace, project, and domain
- Run a crawl and persist technical findings
- Review issues by severity and impact
- Inspect a page-level detail screen
- Connect Search Console and view performance beside crawl data
- Refresh PageSpeed data
- Generate AI explanations and fix suggestions grounded in stored issues

## Deferred

- Billing and limits
- Exportable PDF reports
- Scheduled crawls and alerts
- Public API and webhooks
- White-labeling
- Bing integration
